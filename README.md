

# 🌌 Studio Aura | Landing Page

O **Studio Aura** é uma landing page de alto padrão para um escritório de design de interiores e arquitetura. O projeto materializa o conceito de **"Luxo Discreto"** — uma estética minimalista e cinematográfica que prioriza o silêncio visual, a precisão arquitetônica e a curadoria de materiais.

---

## 🛠️ Ecossistema Tecnológico

Este projeto foi desenvolvido utilizando o estado da arte em automação e inteligência artificial para garantir performance e fidelidade visual:

* **Design & Concepção:**
* **Google Stitch:** Utilizado para garantir a consistência absoluta dos tokens de design.
* **Meta AI:** Geração de assets visuais e fotografias de interiores exclusivas, alinhadas ao briefing de luxo.


* **Desenvolvimento & Automação:**
* **IDE com IA Integrada:** Agilização da escrita de algoritmos e lógica estrutural.
* **npx skills (Taste Skill):** Integração da skill `taste-skill` para padrões estéticos de front-end refinados.


* **Animações de Alta Performance (GSAP Skills):**
* `gsap core`: Motor principal de animação.
* `gsap performance`: Otimização de renderização para garantir 60fps.
* `GSAP ScrollTrigger`: Orquestração de entradas em lote e efeitos de parallax.



---

## 🎨 Design System

O sistema é focado na "confiança silenciosa" de uma galeria de arte:

* **Paleta de Cores:** Fundo charcoal profundo (`#141313`) com acentos em Gold Champanhe (`#C5A059`).
* **Tipografia:** *Playfair Display* (Serif) para headlines e *Montserrat* (Sans-serif) com peso leve para corpo de texto.
* **Formas:** Geometria estritamente **Sharp (0px radius)**, reforçando a integridade estrutural e precisão técnica.

---

## 🚀 Funcionalidades Principais

* **Imersão Cinematográfica:** Vídeos de fundo com `playbackRate` reduzido para criar uma atmosfera de serenidade.
* **Portfólio Dinâmico:** Galeria renderizada via JavaScript com suporte a `lazy loading` e `async decoding`.
* **Experiência de Scroll:** Revelações suaves em lote (*batch reveals*) e micro-interações táteis nos botões.
* **Formulário de Briefing:** Validação inteligente e feedback visual de envio para o cliente de alto padrão.

## ⚙️ Como Executar o Projeto

1. Clone o repositório:
```bash
git clone [https://github.com/seu-usuario/studio-aura.git](https://github.com/seu-usuario/studio-aura.git)

```


2. Instale as dependências de Skill (opcional para desenvolvimento):
```bash
npx skills add [https://github.com/Leonxlnx/taste-skill](https://github.com/Leonxlnx/taste-skill)
npx skills add [https://github.com/greensock/gsap-skills](https://github.com/greensock/gsap-skills)

```


3. Abra o `index.html` com o **Live Server**.
